# PEAF 2026

Website for the **First International Workshop on Pedagogical Evaluation of Automated Feedback (PEAF 2026)**.

The site is built with [Jekyll](https://jekyllrb.com/) and hosted on GitHub Pages at:
**https://peaf-workshop.github.io/2026/**

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Then open http://localhost:4000/2026/

## Pages

| File | Page |

| `index.md` | Home / About |

| `submission.md` | Submission |

| `organizers.md` | Organizers |

## GitHub Pages Setup

In repository Settings → Pages, set source to branch `main`, folder `/ (root)`.
